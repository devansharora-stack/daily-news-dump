More Is Not Always Better: Cross-Component Interference in LLM Agent Scaffolding
arXiv AI
Researchers found that stacking all scaffolding components (planning, tools, memory, self-reflection, retrieval) on LLM agents actually degrades performance — the 'All-In' system was consistently suboptimal across experiments with Llama-3.1 models. This directly challenges the 'more features = better agent' assumption driving enterprise AI agent development, and suggests companies should rigorously test component combinations rather than adding capabilities indiscriminately.
